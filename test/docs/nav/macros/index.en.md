# Macros

[Documentation oficial](https://mkdocs-macros-plugin.readthedocs.io/en/latest/advanced/)
A macro is a parameterized piece of code that we can use as a template.

We need to install the macros plugin:
```sh
pip install mkdocs-macros-plugin
```
